# JUDIAgent Documentation

Start with the root `README.md` for installation and basic usage. This directory
contains focused guides for reproducibility, cluster development, and coding
agent maintenance.

## Guides

- `reproducibility.md`: public-release setup, test commands, benchmark catalog,
  RAG index rebuild notes, and publication artifact guidance.
- `devel-pace.md`: PACE/shared-cluster workflow, including login-node versus
  compute-node responsibilities and cache locations.
- `codex-setup.md`: optional maintainer notes for Codex or similar coding
  agents. Normal users do not need this file.

## Related Files

- `../AGENTS.md`: canonical guidance for AI coding agents working in this repo.
- `../judiagent_tutorial/README.md`: tutorial notebook setup.
- `../NOTICE`, `../CITATION.cff`, and
  `../src/judiagent/rag/THIRD_PARTY.md`: public-release attribution and
  third-party corpus notes.
