# Abstract

We present **JUDIAgent**, a JUDI-specific scientific coding agent for seismic modeling, imaging, and inversion workflows. Rather than treating large language models as standalone code generators, JUDIAgent couples retrieval-augmented generation with a dual-validation loop tailored to wave-equation-based geophysics. Given a natural-language task, the agent retrieves relevant JUDI documentation and examples, synthesizes Julia code, checks executable correctness through lint and runtime validation, and then performs a second domain-aware review that evaluates whether the generated workflow is scientifically complete for the target seismic task.

The key design choice is a **dual validation architecture**. The first stage focuses on software correctness: syntax, imports, executable Julia structure, and runtime behavior. The second stage focuses on scientific adequacy: whether the generated code includes the expected ingredients of a credible seismic workflow, such as model setup, acquisition geometry, forward or inversion operators, and quality diagnostics. JUDIAgent also attaches **task-aware metric guidance**, for example image residual and illumination metrics for migration tasks or objective decrease and gradient update norms for inversion tasks. This makes the system useful not only for interactive coding assistance but also as a foundation for structured supervision, benchmark design, and future reward modeling.

The current framework supports two complementary agent modes: an iterative code-generation agent with mandatory validation and an autonomous ReAct-style agent for open-ended tool use. Together they provide a practical path from natural-language user intent to validated JUDI workflows. We argue that JUDIAgent is best understood not as a generic evaluator-optimizer clone, but as a **domain-grounded scientific coding system** whose main contribution is the integration of software validation with seismic workflow validation. This framing is especially relevant for future work on SFT, RLHF, and benchmark-based evaluation of scientific coding agents.
